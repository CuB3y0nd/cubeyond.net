## Running Locally

```bash
git clone https://github.com/CuB3y0nd/cubeyond.net.git
cd cubeyond.net
```

Then install dependencies and run the development server:

```bash
pnpm install
pnpm dev
```

## Building for Production

```bash
pnpm build
```
